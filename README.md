### 简介
💖该仓库是用来对文档中表格做结构化识别的推理库，表格结构使用的是自训练模型。相关代码是基于[TableStructureRec](https://github.com/RapidAI/TableStructureRec/tree/main) 进行的二次开发。

### 安装

